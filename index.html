Conditions

What is the output of the following java code?


							int i = 0; int j = 1;
							if (i == 1)
								if (j==1)
						        		System.out.println("j == 1");
						        else
						         	System.out.println("i != 1");

Conditions

Answer: Nothing.

Reason: The else belongs to the inner if

Conditions

Solution

Always write parentheses in if statements.


						   int i = 0; int j = 1;
						   if (i == 1) {
						   	if (j==1) {
						        	System.out.println("j == 1");
							}
						   	else
						   	{
						   		System.out.println("i != 1");
						   	}
						   }
